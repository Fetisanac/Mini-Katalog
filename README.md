# Mini Katalog Uygulaması

## Kısa Açıklama
Bu proje, Flutter eğitimi kapsamında geliştirilmiş temel bir mobil e-ticaret/katalog uygulamasıdır. Projenin temel amacı; widget mimarisi, `GridView` ile kart tabanlı UI tasarımı, sayfa geçişleri (`Navigator`), sayfalar arası veri taşıma (`Route Arguments`) ve basit state yönetimi kavramlarını pratik etmektir. Herhangi bir harici paket kullanılmamış olup (`material.dart` temellidir), API kullanımı simüle edilerek ürün verileri JSON formatından çekilerek listelenmiştir.

## Kullanılan Flutter Sürümü
* **Flutter SDK:** 3.44.3 (Channel stable)
* **Dart SDK:** 3.4.3

## Çalıştırma Adımları
Projeyi kendi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

1. **Projeyi Klonlayın:**
   ```bash
   git clone [https://github.com/Fetisanac/Mini-Katalog.git](https://github.com/Fetisanac/Mini-Katalog.git)
##Proje Dizinine Gidin:


cd mini_katalog
Gerekli Bağımlılıkları Yükleyin:


flutter pub get
Uygulamayı Çalıştırın:
Bağlı bir emülatör veya fiziksel cihaz üzerinde uygulamayı başlatmak için:
